# create-react-app-ts template

## Quick start

```bash
   $ yarn
   $ yarn dev  
```

## Include

- create-react-app 体验
- 集成typescript与tslint，tslint的规则与杭州前端团队一致
- 配置本地开发环境，后端及杭州请求会转发到 i18n.qunhequnhe.com
- 默认ui组件库为antd
- 已预置coohom iconfont，使用时可以以 ```<MyIcon type="xxx"/>```方式引入
- utils内为Gateway的axios instance，解决请求携带jwt的问题。
- 预置react-router
